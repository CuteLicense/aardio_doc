# 算术运算符

 对两个操作数进行算术运算如果其中是一个操作数是字符串则尝试转换为数值运算，转换失败则抛出异常。

如果是其他类据类型则查找对象的元方法进行运算，如果对象并未定义元方法则抛出异常。

!> ![](../../icon/warning.gif) aardio会将引号前后的+运算符自动转换为字符串[连接运算符](the%20language/operator/concat) ++

|  运算符 |  说明 |
| --- | --- |
|  + |  加 |
| - |  减(二元运算符)、取负(单目运算符) |
|  * |  乘 |
| / |  除 |
|  ** |  幂 |
| % |  模 |
